# Business Flow Creator

業務フローを視覚的に作成・編集できるWebアプリケーションです。

## 機能

### 主要機能
- **フロー図作成**: グリッドベースのキャンバス上でフローチャートを作成
- **要素管理**: カラーカード、矢印、分岐記号などの要素を管理
- **テキスト編集**: フロー要素内のテキストを編集可能

### 副次機能
- フロー保存・読み込み機能（LocalStorage）
- フロー画像エクスポート機能（JSON形式）
- グリッド表示/非表示切り替え
- グリッドスナップ機能
- アンドゥ/リドゥ機能（Deleteキーで削除）

## 使用方法

1. **要素の配置**
   - 左側のツールパレットから要素を選択
   - キャンバス上でクリックして要素を配置

2. **要素の編集**
   - 要素をクリックして選択
   - 右側のプロパティパネルでテキスト、カラー、サイズを編集

3. **要素の移動**
   - 要素をドラッグして移動
   - グリッドスナップ機能で整列

4. **ファイル操作**
   - 保存: 自動的にLocalStorageに保存
   - 読み込み: ページ読み込み時に自動復元
   - エクスポート: JSON形式でダウンロード
   - クリア: すべての要素を削除

## 技術仕様

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **UIフレームワーク**: Bootstrap 5
- **データ管理**: LocalStorage API
- **グラフィック**: HTML5 Canvas
- **ブラウザ対応**: Chrome, Firefox, Safari, Edge

## ファイル構成

```
business_flow/
├── index.html          # メインHTMLファイル
├── styles.css         # カスタムCSS
├── script.js          # メインJavaScript
└── docs/
    └── 要件定義.md     # 要件定義書
```

## 起動方法

1. `index.html`をブラウザで開く
2. または、ローカルサーバーでホスティング

```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```
